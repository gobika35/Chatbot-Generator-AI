# Chatbot Generator AI

A production-ready backend that generates **business-specific chatbots**
with intent locking, lead capture, and admin-only theming.

## Features
- Groq-powered LLM (LLaMA / Mixtral)
- Intent detection & locking
- Business-specific system prompts
- Lead storage for human assistants
- Admin-only chatbot theme customization
- Preview chatbot API
